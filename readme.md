##Process Tweets in Real-Time with Laravel
Streaming API: There are three Streaming APIs, but the one we care about is the Public API - and more specifically, the 'filter' endpoint. This endpoint will deliver a stream of all public tweets, filtered by the keywords that you define.
Phirehose: The Phirehose library takes care of authentication and other implementation details for the Streaming API.
